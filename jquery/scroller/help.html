1. Overview:

This is a jQuery image scroller with lightbox. The scroll bar/indexes and directional buttons allow for easy navigation of your image gallery. Image click can either open the included lightbox or a regular link. The scroller is also re-sizable and fully configurable through the plugin's parameters.

Features:
  • Configurable number of slides visible per scroll. (i.e. the demo is showing 3 slides per scroll)
  • Can turn auto scroll rotation on/off. Auto scroll is paused on mouseover.
  • Images can be scaled to fit in slide window.
  • Configurable text caption, including positioning inside/outside and top/bottom.
  • Captions allow for html description, including embedding links.
  • Able to add unlimited number of images.
  • Use scroll bar to navigate. Images auto stop to its nearest position. Also, scroll bar's track is clickable.
  • Alternatively, use the indexes instead of scroll bar for navigation.
  • Can set images in random order.
  • Touchscreen and mousewheel scroll supported.
  • Lightbox plugin included, can be used seperately from scroller on any links.
  • Lightbox's rotation is available.
  • Lightbox is automatically positioned and fitted accordingly to browser window.
  • Lightbox supports image, iframe, ajax, flash, and inline content.

2. Installation:

Installing the scroller within your html page requires the following steps.

  1. Including the scroller's and lightbox's css files.
  2. Including the required javascript files.
  3. Specifying the html tags of the scroller within your page.
  4. Instantiating a javascript function call with any optional parameters.

Below describes these steps in more detail.

In step 1, to include the css files wt-scroller.css and wt-lightbox.css, simply specify the following lines of code within your <head> tag: 

<link rel="stylesheet" type="text/css" href="wt-scroller.css"/>
<link rel="stylesheet" type="text/css" href="lightbox/wt-lightbox.css"/>

In step 2, you will need to include the jquery's library file, jquery-1.7.1.min.js and jquery's UI file, jquery-ui-1.8.10.custom.min.js. Also, you will need to include the scroller's js file named jquery.wt-scroller.min.js and the optional lightbox's js file, jquery.wt-lightbox.min.js if you plan on using the lightbox. All files are located within the js directory.
The code below shows how to include them in a html page: 

<script type="text/javascript" src="js/jquery-1.7.1.min.js"></script>
<script type="text/javascript" src="js/jquery-ui-1.8.10.custom.min.js"></script>
<script type="text/javascript" src="lightbox/js/jquery.wt-lightbox.min.js"></script>
<script type="text/javascript" src="js/jquery.wt-scroller.min.js"></script>

note: the js files with min in its file names are the compressed production versions of the files. If you intend to modify the source code, you will need to look at the non-min development versions of the files and it is recommended that you minify the modified files afterwards.

In step 3, to specify the tags of the scroller itself, refer to the example in the index.html file. In that file, you'll see a block of tag starting and ending with: 

<div class="container">
   <div class="wt-scroller">
      ...
   </div>
</div>

You can use this as a template and copy and paste it onto your own page, but will have to modify it accordingly with your own image information. Section 3 will describe this step in detail.

Finally in step 4, you will need to instantiate a javascript function call to get the scroller initialized. An additional javascript function call to initial the lightbox is needed only if the lightbox is used. The example code looks like this below: 

<script type="text/javascript">
$(document).ready(
   function() {
      //initialize scroller
      $(".container").wtScroller();
      
      //initialize lightbox for scroller
      $("a[rel='scroller']").wtLightBox();
   }
);
</script>

This will instantiate the scroller and lightbox using the default parameters. Section 5 will describe instantiation with custom parameters.

3. Adding Content:

In order to add or change content to the scroller, you will have to specify a set of tags for each image within the scroller's html tags. The list of images is represented as a list tag <ul> and each image is represented by a list item tag <li>.

Below is an example of what the tags would look like for an image. You will have to repeat this for as many images as needed for your scroller. 

<li>
    <a href="images/image1.jpg" rel="scroller">
        <img src="images/thumbs/thumb1.jpg"/>
    </a>    
    <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Ut consequat.</p>
</li>

The example above shows that each image entry begins with an opening and closing <li> tag. Within the <li> tag, there is an <a> tag specifying an image's url as its href value. This is the image url use for the lightbox if lightbox is used. Also, the attribute rel within the <a> tag is used purely to mark links for the lightbox. The value for rel can be anything as long as it matches the lightbox initialization call as shown later in Section 5.

If you are not planning on using the lightbox, you can specify instead a regular link to be opened when the image is clicked. You can also specify a link target here. In addition, if you do not wish to have a link associated with the image, you can omit this <a> altogether.

Next, within the <a> tag, there is an <img> representing the slide's image.

And finally, the <p> tag represents the caption associated with the slide. You can specify text description within the tags. If you do not want a caption with a particular slide, you can omit the <p> tag.

4. Adding Different Lightbox Content:

This section will describe how to add different types of content for the lightbox. Currently, the lightbox supports image, flash, iframe, ajax, and inline content.

Since the lightbox works on links (<a> tags), you will need to assign a value to the href attribute as to where to locate the specific content.

First, to add an image to the lightbox, you will need to assign the url of your image to the href attribute of the <a> tag.

Below shows an example of using image for the lightbox. 

<a href="images/example.jpg">
    <img src="images/thumbs/thumb1.jpg"/>
</a>

To add flash content to the lightbox, you will need to assign the url of your swf file to the href attribute of the <a> tag. You can also assign a specific width and height value by using the url variable of boxwidth and boxheight.

Below shows an example of using flash for the lightbox with a set width and height. 

<a href="swf/example.swf?boxwidth=800&boxheight=600">
    <img src="images/thumbs/thumb1.jpg"/>
</a>

To add inline content to the lightbox, you will need to assign the id name to the href attribute of the <a> tag. The id name should reference the id of a tag in your page where you want the html content to come from. Again, you can assign a width and height value by using the url variable of boxwidth and boxheight.

Below shows an example of using inline content for the lightbox. Be sure to include # sign in front of the id name. 

<a href="#exampleid?boxwidth=800&boxheight=600">
    <img src="images/thumbs/thumb1.jpg"/>
</a>

To add an iframe to the lightbox, you will need to assign an url to the href attribute of the <a> tag. Also, you need to specify an url variable named boxtype and assign it the value iframe. Again, you can assign a width and height value by using the url variable of boxwidth and boxheight.

Below shows an example of using iframe for the lightbox. 

<a href="http://www.codecanyon.net?boxtype=iframe&boxwidth=800&boxheight=600">
    <img src="images/thumbs/thumb1.jpg"/>
</a>

Finally, to add an ajax page to the lightbox, you will need to assign your ajax page url to the href attribute of the <a> tag. Also, you need to specify an url variable named boxtype and assign it the value ajax. Furthermore, you can add other request variables at the end of the url for your ajax page to process.

Below shows an example of using ajax for the lightbox. 

<a href="example.php?boxtype=ajax&boxwidth=800&boxheight=600&var1=abc&var2=def">
    <img src="images/thumbs/thumb1.jpg"/>
</a>

5. Plugin Parameters:

You can change some common attributes of the scroller and lightbox through their plugin parameters. Below is an example of the attributes being specified with corresponding values. 

<script type="text/javascript">
$(document).ready(	
    function() {
    	//initialize scroller
        $(".container").wtScroller({
            num_display:3,
            slide_width:300,
            slide_height:200,
            slide_margin:1,
            button_width:35,
            ctrl_height:25,
            margin:10,    
            auto_scroll:true,
            delay:4000,
            scroll_speed:1000,
            easing:"",
            auto_scale:true,
            move_one:false,
            ctrl_type:"scrollbar",
            display_buttons:true,
            mouseover_buttons:false,
            display_caption:true,
            mouseover_caption:false,
            caption_effect:"slide",            
            caption_align:"bottom",
            caption_position:"inside",                    
            cont_nav:true,
            shuffle:false,
            mousewheel_scroll:true
        });
		
        //initialize lightbox for scroller	
        $("a[rel='scroller']").wtLightBox({
            rotate:true,
            delay:4000,
            duration:600,
            display_number:true,
            display_dbuttons:true,
            display_timer:true,
            display_caption:true,
            caption_position:"outside",
            cont_nav:true,
            auto_fit:true,
            easing:"",
            type:"GET"
        });
    }
);
</script>

note: if you want to leave a parameter with its default value, you can omit that parameter in the javascript call.

The table below describes each attribute in more detail:

Scroller's Parameters Table:

Attribute Name Description Default Value Possible Values
num_display The number of slide displayed at a time. 3 Any positive number.
slide_width The width of each slide. 300 Any positive number.
slide_height The height of each slide. 200 Any positive number.
slide_margin The margin between each slide. 1 Any number.
button_width The width of the directional buttons. 35 Any positive number.
ctrl_height The height of the scrollbar or index bar. 25 Any positive number.
margin The margin around the scroller.
note: The left and right margin is not use if the directional buttons are displayed. Also, the bottom margin is not use if either the scrollbar or index bar is displayed. 		10 Any number.
auto_scroll Specify if auto scroll rotation is on. true true, false.
delay The time delay in milliseconds for auto scroll rotation. 4000 Any positive number.
scroll_speed The scrolling speed in milliseconds. 1000 Any positive number.
easing The easing used for scrolling transition. "" Please visit jQuery Easing Plugin for a list of all available easing.
auto_scale Specifies if slide images are automatically scaled to fit centerly within slide window. false true, false.
move_one Specifies if slides move by one per scroll.
note: Move by one does not apply when ctrl_type is set to index.		 false true, false.
ctrl_type Specifies the type of navigation control used for the lower panel of the scroller. "scrollbar" "scrollbar", "index", "none"
display_buttons To specify if the directional buttons are visible. true true, false.
mouseover_buttons To specify if the directional buttons appears on mouseover and hides on mouseout. false true, false.
display_caption To specify if the slide captions are visible. true true, false.
mouseover_caption To specify if the captions appears on mouseover and hides on mouseout.
note: Mouseover only works if caption_position is inside.		 false true, false.
